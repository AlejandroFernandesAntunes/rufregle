rufregle-translate
==================

Simple unoficial free google translate gem.

Install and Run
====

Tested only in Ruby version 2.1.2

Installing gem
```
 gem install rufregle

```

Using on code:
```

    result = Rufregle.translate("Hello World", "en", "pt")
    p "#{result[0]}"	


```
